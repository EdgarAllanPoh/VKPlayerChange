# VK Video Live - Смена плеера
Полный бредик на случай если вы хотите смотреть стрим с нормальным плеером, но оставаться при этом в чате VK
## Установка
1. Установите расширение Tampermonkey: https://www.tampermonkey.net/
2. Нажмите [сюда для установки](https://raw.githubusercontent.com/EdgarAllanPoh/VKPlayerChange/main/VKPlayerChange.user.js) (Если не открывается, отключите/включите VPN)
3. Нажмите "Установить"

## Использование
- Откройте любой стрим на VK Видео Live
- Нажмите кнопку "Сменить плеер"
- Выберите платформу и введите канал(на Youtube - ссылка, на Kick и Twitch - Название канала)
- При обновлении страницы или переходе на другие каналы, плеер сбрасывается

## Возможности
- YouTube, Twitch, Kick
- Кнопка возврата к VK плееру


## Если не отображается кнопка после установки 
Нужно включить "Режим разработчика" в браузере - https://www.tampermonkey.net/faq.php?q=Q209#Q209
